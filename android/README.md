# Buzz Android — scaffold

Kotlin/Compose Android app, sharing the Supabase backend with the iOS multiplatform app
and the Next.js web app. **This is a scaffold, not a ship-ready Android app.** It
compiles, launches, and shows a placeholder feed; full feature parity with iOS is the
next phase.

## Setup (one-time)

1. Open the `android/` directory in **Android Studio** (Hedgehog 2024.3.1+ recommended).
2. Wait for the Gradle sync to complete — the wrapper (`gradlew`) is generated by Studio.
3. Create `android/local.properties` (gitignored) and add your Supabase credentials:
   ```
   sdk.dir=/Users/<you>/Library/Android/sdk
   BUZZ_SUPABASE_URL=https://<your-project>.supabase.co
   BUZZ_SUPABASE_ANON_KEY=<your-anon-key>
   ```
4. Pick an emulator (Pixel 8 API 35 is a good default) and **Run**.

## Stack

- Kotlin 2.0.21, AGP 8.7.3, JDK 17, minSdk 26 / target+compileSdk 35
- Jetpack Compose (BOM 2024.12.01) + Material 3
- Navigation Compose
- Supabase-kt 3.0.3 — Postgrest + Auth + Realtime
- Ktor 3.0.2 (OkHttp engine)
- Coil 2.7.0
- DataStore (preferences)
- kotlinx-serialization

## Parity with iOS

| Concept | iOS | Android |
|---|---|---|
| Application id | `com.arnavgoel.buzz` | `com.arnavgoel.buzz` |
| Universal links / app links | `applinks:buzz.app` | `https://buzz.app` (`autoVerify=true`) |
| Default appearance | Forced dark (`UIUserInterfaceStyle: Dark`) | Dark by default, light fallback for hygiene |
| Backend | Supabase | Supabase (same project) |
